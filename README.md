# COMP 473 - Fall 2019

## Assignment 1

### How to run

1. Install requirements: 
```
pip install -r requirements.txt
```
2. Run script: 
```
path/to/python3 classifier.py
```

To output the results to a file, run:
```
path/to/python3 classifier.py > results.txt
```
